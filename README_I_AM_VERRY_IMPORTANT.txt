1.6
=================================================================================================================
à cause d'un bug de compatibilité (Blur/TACZ),
il est reccomandé de télécharger et installer le patch (contenu dans le dossier patch ) nommé blur-client.toml
pour l'installer :

	- copier le ficher contenu dans le dossier "patch" du .zip
 
	- aller dans votre dossier de configuration (normalement localisé dans "%appdata%\.minecraft\config" pour les modpacks normaux et 
	 "C:\Users\<nom d'utilisateur windows>\curseforge\minecraft\install\<nom du modpack>\config" pour les utilisateurs CurseForge)
	 OU, version plus simple, appuyer sur ouvrir le dossier dans le launcher.

	- remplacer "blur-client.toml" par la version fournis

à noter que ce patch est optionnel. c'est juste mieux de l'installer et que ce fichier peut être remplacé si le jeu est en cours d'exécution
