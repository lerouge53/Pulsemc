1.6
=================================================================================================================
à cause d'un bug de compatibilité (Blur/TACZ),
il est reccomandé de télécharger et installer le patch (contenu dans le dossier patch ) nommé blur-client.toml
pour l'installer :

	- copier le ficher contenu dans le dossier "patch" du .zip
 
	- aller dans votre dossier de configuration (normalement localisé dans "%appdata%\.minecraft\config" pour les modpacks normaux et 
	 "C:\Users\<nom d'utilisateur windows>\curseforge\minecraft\install\<nom du modpack>\config" pour les utilisateurs CurseForge)
	 OU, version plus simple, appuyer sur ouvrir le dossier dans le launcher.

	- remplacer "blur-client.toml" par la version fournis

à noter que ce patch est optionnel. c'est juste mieux de l'installer et que ce fichier peut être remplacé si le jeu est en cours d'exécution

Vous avez probablement remarqué le nouveau dossier (appelé "kubejs") dans le zip d'instalation. il est important.

==================================================================================================================
2.1
==================================================================================================================
il y a pas mal de fichier qui on été rajouté. Ils sont quasiment tous des fichiers d'instalation automatique. 
certains des fichiers rajouté sont des scripts d'exécution powershell (.ps1). Il ne peuvent pas être double-click (ouvre le bloc-note).
si vous voulez le lançer, il faudra double-click sur le script de commande de votre système d'exploitation (.cmd sur windows et .sh sur linux (vous devez installer powershell si vous voulez utiliser le programme d'installation sur linux). MacOS n'est pas encore supporté).
Il est recommandé de le lançer (juste parce que c'est plus facile et rapide) mais vous êtes libre de faire l'instalation manuellement.
aussi, pour les personnes qui ont installé le modpack avant la 2.1, vous DEVEZ impérativement rajouter à la racine de votre dossier minecraft (au même endroit que le dossier mods, save, etc...) le fichier pulsemc.installed.flag sans aucune autre donnée à l'intérieur.
